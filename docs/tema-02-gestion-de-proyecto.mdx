# Tema 2 - Gestion de proyecto

## Objetivos

- Planificar iteraciones realistas para un equipo de desarrollo.
- Definir roles, responsabilidades y flujos de comunicacion.
- Medir avance y tomar decisiones de ajuste.

## Contenidos

- Alcance, backlog y criterios de prioridad.
- Estimacion de tareas y planificacion por sprint.
- Gestion de riesgos y dependencias.
- Indicadores de seguimiento (velocidad, burndown, hitos).

## Actividades

1. Crear backlog inicial y priorizacion MoSCoW.
2. Planificar dos iteraciones con objetivos medibles.
3. Configurar tablero de seguimiento del proyecto.
4. Informe de riesgos con acciones de mitigacion.

## Evaluacion

- Realismo de la planificacion.
- Trazabilidad entre objetivos y tareas.
- Capacidad de reaccion ante desviaciones.

## Recursos

- Plantilla de backlog y roadmap.
- Ejemplos de rituales de seguimiento.
- Guia de comunicacion de equipo.
