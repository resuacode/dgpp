# Guía de estilo de contenido

> **Documento interno.** El prefijo `_` hace que Docusaurus excluya este archivo del sitio
> publicado: no aparece en la web ni en el sidebar. Uso exclusivo del equipo docente.

## Objetivo

Esta guía define cómo redactar material para mantener consistencia didáctica y técnica en todo el portal.

## Tono y claridad

- Escribir en español claro, con frases directas.
- Priorizar verbos de acción y resultados observables.
- Evitar párrafos excesivamente largos.

## Estructura recomendada por tema

1. Objetivos.
2. Contenidos.
3. Actividades.
4. Evaluación.
5. Recursos.

## Buenas prácticas MDX

- Usar encabezados jerárquicos (`##`, `###`) con nombres descriptivos.
- Incluir bloques de código solo cuando aporten al aprendizaje.
- Usar listas numeradas para secuencias de trabajo.

## Criterios pedagógicos

- Cada actividad debe indicar evidencia esperada.
- Cada criterio de evaluación debe ser comprobable.
- Cada tema debe conectar con contexto profesional.

## Convenciones de archivos

- Nombres en minúsculas con guiones.
- Sin espacios ni acentos en nombres de fichero.
- Assets en `static/img/` o en subcarpetas temáticas según necesidad.
