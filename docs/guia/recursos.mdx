## Herramientas recomendadas

Sugerencias por función. No son obligatorias: el equipo elige según el itinerario y su
experiencia, pero debe justificar la elección.

| Función | Opciones habituales |
| --- | --- |
| Motor de juego | Unity, Unreal Engine, Godot (según itinerario) |
| Control de versiones | Git + GitHub / GitLab |
| Gestión ágil | Trello, GitHub Projects, Jira, Notion |
| Documentación | Markdown/MDX, Google Docs, Notion |
| Diagramas | draw.io, Excalidraw, Figma |
| Hojas de cálculo | Google Sheets, LibreOffice Calc, Excel |
| Captura de gameplay | OBS Studio, captura del motor, ShadowPlay |
| Edición de vídeo | DaVinci Resolve, Shotcut, CapCut |
| Audio y música con licencia | Bibliotecas del motor, Freesound (CC), Artlist |

:::tip[Herramientas para diseñadores dentro del motor]
Más allá de las herramientas externas, el módulo trata la creación de **herramientas de
producción propias** (editor scripting) para que diseñadores no programadores trabajen dentro
del motor. Se desarrolla en el
[Tema 5.1 — Level design y editor scripting](../tema-05/tema-05-level-design-e-editor-scripting.mdx).
:::
