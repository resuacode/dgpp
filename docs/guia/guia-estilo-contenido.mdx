# Guia de estilo de contenido

## Objetivo

Esta guia define como redactar material para mantener consistencia didactica y tecnica en todo el portal.

## Tono y claridad

- Escribir en espanol claro, con frases directas.
- Priorizar verbos de accion y resultados observables.
- Evitar parrafos excesivamente largos.

## Estructura recomendada por tema

1. Objetivos.
2. Contenidos.
3. Actividades.
4. Evaluacion.
5. Recursos.

## Buenas practicas MDX

- Usar encabezados jerarquicos (`##`, `###`) con nombres descriptivos.
- Incluir bloques de codigo solo cuando aporten al aprendizaje.
- Usar listas numeradas para secuencias de trabajo.

## Criterios pedagogicos

- Cada actividad debe indicar evidencia esperada.
- Cada criterio de evaluacion debe ser comprobable.
- Cada tema debe conectar con contexto profesional.

## Convenciones de archivos

- Nombres en minusculas con guiones.
- Sin espacios ni acentos en nombres de fichero.
- Assets en `static/img/` o en subcarpetas tematicas segun necesidad.
