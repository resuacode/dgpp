# Material del módulo DGPP

Este sitio recopila el material del módulo **Diseño, Gestión, Publicación y Producción de Videojuegos** del curso de especialización de FP en desarrollo de videojuegos.

## Cómo usar este portal

1. Revisa la [Guía docente](./guia-docente.mdx) para conocer enfoque, metodología y evaluación.
2. Avanza por los temas del sidebar en orden recomendado.
3. Completa las actividades de cada tema y sube las evidencias solicitadas.
4. Consulta [Recursos](./recursos.mdx) para ampliar o reforzar contenidos.

## Estructura del contenido

- Bloque 1: Diseño y preproducción.
- Bloque 2: Gestión de proyecto.
- Bloque 3: Publicación y marketing.
- Bloque 4: Producción y pipeline.

## Convención de cada tema

Cada tema incluye:

- Objetivos de aprendizaje.
- Contenidos clave.
- Actividades propuestas.
- Criterios de evaluación.
- Recursos complementarios.
