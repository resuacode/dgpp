# Material del modulo DGPP

Este sitio recopila el material del modulo **Diseno, Gestion, Publicacion y Produccion de Videojuegos** del curso de especializacion de FP en desarrollo de videojuegos.

## Como usar este portal

1. Revisa la [Guia docente](./guia-docente.mdx) para conocer enfoque, metodologia y evaluacion.
2. Avanza por los temas del sidebar en orden recomendado.
3. Completa las actividades de cada tema y sube las evidencias solicitadas.
4. Consulta [Recursos](./recursos.mdx) para ampliar o reforzar contenidos.

## Estructura del contenido

- Bloque 1: Diseno y preproduccion.
- Bloque 2: Gestion de proyecto.
- Bloque 3: Publicacion y marketing.
- Bloque 4: Produccion y pipeline.

## Convencion de cada tema

Cada tema incluye:

- Objetivos de aprendizaje.
- Contenidos clave.
- Actividades propuestas.
- Criterios de evaluacion.
- Recursos complementarios.
